About ConnectSphere

ConnectSphere is a modern social media platform built with Django. It allows users to create profiles, post updates, engage through likes and comments, and chat via real-time messaging.

Key Features

  • User Authentication (Signup/Login)
  • Profile Customization
  • Post Creation, Likes, and Comments
  • Real-time Messaging

Technologies Used

Backend: Django (Python)
Frontend: HTML/CSS/JS
Database: SQLite3
Real-time: WebSockets
Auth: Django Auth

Developed with a focus on user engagement and security, ConnectSphere provides an intuitive and seamless experience for online interaction.
